# Botato 🥔

> A playful local LLM chatbot for private, offline use.
> No internet. No API costs. Just you and your potato. 🥔

## Stack

- Node.js + TypeScript
- React (Frontend)
- Hono (Backend Server)
- LangChain + Ollama (llama3.1)
